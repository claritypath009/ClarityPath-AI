## Problem Statement
Learners waste time choosing what to learn next. They often feel overwhelmed by the number of courses, videos, and resources online, and struggle to create a structured learning path that matches their goals and current skills.

## Goals
- Provide **personalized learning paths** for each user
- Prioritize **quality resources** (free or paid) based on relevance and effectiveness
- Maintain **trust and transparency** in recommendations
- Support students, fresh graduates, self-learners, career switchers, working professionals, and companies training staff

## Target Users
- Students
- Fresh graduates
- Self-learners
- Tech career switchers
- Working professionals
- Companies training staff

## Key Features
- AI-generated personalized learning paths
- Skill gap analysis
- Progress tracking
- Option to filter by free or paid resources
- Transparent recommendations and disclaimers

## Workflow
1. User enters goals, current skills, and preferences
2. AI evaluates skill gaps
3. AI generates a personalized learning path
4. User reviews roadmap and weekly tasks
5. User tracks progress and optionally saves/exports

## Competitor Analysis
| Competitor | Strength | Gap / Opportunity |
|------------|---------|-----------------|
| Coursera / Udemy | Course recommendations | No structured AI-generated learning paths |
| Skillshare | Topic suggestions | No goal-aligned roadmap |
| EdApp / Degreed | Corporate learning & analytics | Complex, not beginner-friendly |
| Pathrise / CareerFoundry | Mentor-guided paths | Human-led, not automated |

## User Stories & Acceptance Criteria
**User Story 1:** As a learner, I want a learning path based on my goals and skills so that I don’t waste time choosing resources.  
- **Acceptance Criteria:** AI provides at least 3 relevant resources per skill area.

**User Story 2:** As a learner, I want to filter by free or paid resources so I can choose based on my budget.  
- **Acceptance Criteria:** Filter works and updates recommendations in real time.

**User Story 3:** As a company manager, I want to track staff learning progress so I can assess training effectiveness.  
- **Acceptance Criteria:** Progress dashboard updates correctly and shows completion percentage.

## Feasibility / Cost
- Prototype: Figma, Notion, Glide → minimal cost  
- AI recommendation: free-tier OpenAI API or similar  
- Full deployment: API usage, hosting, and possible premium subscriptions may incur cost
